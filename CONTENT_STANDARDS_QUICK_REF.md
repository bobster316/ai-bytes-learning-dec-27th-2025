## ✅ NEW STANDARDS SUMMARY

### **LESSON CONTENT:**

| Level | Words | Images | Age Range | Assumptions |

| **Intermediate** | **900+** | **4** | 14-60 | Basic |
| **Advanced** | **1000+** | **4** | 18-60 | Strong |

---

### **TOPIC DESCRIPTIONS:**
```
Format: 3-4 short paragraphs
Each: 3-4 sentences
Spacing: Blank line between each
Length: 150+ words
HTML: <p> tags with \n\n between
```

---

### **PARAGRAPH RULES:**
- ✅ **2-3 sentences** maximum per paragraph
- ✅ **Blank line** between every paragraph  
- ✅ **15-20 words** per sentence (beginner)
- ✅ **Short + Sweet** = Better readability

---

## 🎯 BEGINNER LEVEL (Ages 10-60)

### **Critical Rules:**
1. ✅ Assume **ZERO** prior knowledge
2. ✅ Explain **EVERY** technical term
3. ✅ Use **everyday analogies**
4. ✅ Think: "Could a **10-year-old** understand?"
5. ✅ **No jargon** without explanation
6. ✅ **Concrete examples** from daily life

### **Example Analogy:**
```
❌ "Neural networks use backpropagation algorithms"
✅ "Think of learning to ride a bike. You make mistakes 
   and learn from them. Neural networks do the same!"
```

## 📝 TOPIC DESCRIPTION TEMPLATE

```html
<p>First paragraph (3-4 sentences) introducing the topic 
and why it matters. Grab attention and set context.</p>

<p>Second paragraph (3-4 sentences) explaining key concepts 
that will be covered. What will students learn?</p>

<p>Third paragraph (3-4 sentences) on practical applications 
and real-world relevance. Why should they care?</p>

<p>Optional fourth paragraph (3-4 sentences) with learning 
outcomes or what they'll be able to do after.</p>
```

## 🔢 WORD COUNT BREAKDOWN

### **Beginner (800+ words):**
- Introduction: ~200 words
- Main content (6 sections): ~450 words
- Hands-on lab: ~80 words
- Other sections: ~70 words
- **Total: 800+ words**

### **Intermediate (900+ words):**
- More depth in each section
- More complex examples
- Additional details

### **Advanced (1000+ words):**
- Deep theoretical insights
- Research-level content
- Comprehensive coverage

## ✅ QUALITY CHECKLIST

### Before Publishing Lesson:
- [ ] Word count: 800-1000+ (based on difficulty)
- [ ] Images: Exactly 4 educational diagrams
- [ ] Paragraphs: 2-3 sentences + spacing
- [ ] Technical terms: Explained (beginner)
- [ ] Analogies: Used (beginner)
- [ ] Age-appropriate: 10+ (beginner)
- [ ] British English: Colour, optimise, programme
- [ ] Line spacing: Between all paragraphs

- [ ] Description: 3-4 paragraphs
- [ ] HTML formatting: `<p>` tags
- [ ] Spacing: `\n\n` between paragraphs
- [ ] Length: 150+ words
