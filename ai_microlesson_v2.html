🎯 Learning Objective
By the end of this module, you will be able to explain the three types of machine learning and identify which type is being used in a real-world AI application in your own industry.

The Core Idea

Traditional software follows rules you write. Machine learning flips this completely. Instead of writing rules, you feed the machine thousands of examples and let it figure out the patterns itself.

Machines learn by processing vast amounts of data and detecting statistical patterns — not by following hand-coded rules.
💡
Key Insight
A spam filter was never programmed to recognise "Buy now! Click here!" as spam. It was trained on millions of emails and discovered those patterns itself.

The Three Types of Machine Learning

Supervised
🏷️ Supervised Learning
Learns from labelled examples. Show it 10,000 photos labelled "cat" or "dog" — it classifies new photos.
Unsupervised
🔍 Unsupervised Learning
Finds hidden patterns without labels. No guidance — the machine discovers structure by itself.
Reinforcement
🎮 Reinforcement Learning
Learns through trial, error, and rewards. Like training a dog — reward good, signal bad.

Understanding It Through Analogy

The Human Parallel
How a child learns to recognise a dog
A parent points and says "dog!" roughly 1,000 times across different breeds, sizes, and settings. The child abstracts a concept from examples — no rulebook required.
The Machine Parallel
How a model learns the same concept
A dataset of 50,000 labelled images is fed to the model. Backpropagation adjusts weights until errors shrink. The model abstracts the same concept — from numbers, not experience.
⚠️
Where the analogy breaks down
A child learns from 3 examples. A machine often needs thousands or millions. This data hunger is one of the biggest practical challenges in deploying real-world AI.

How This Applies to Your Industry

Machine learning is solving specific, concrete challenges across every sector. Select your industry:

AI-assisted medical imaging, St. Thomas Hospital, London
🏥 Healthcare
Supervised learning analyses medical images. Radiologists labelled 100,000 X-rays as "tumour present" or absent. The model now detects certain cancers 11% more accurately than human radiologists — not replacing doctors, but giving them a second pair of eyes that never tires.
Real-time transaction monitoring, London financial district
💳 Finance
Unsupervised learning clusters transaction patterns. Banks never told the algorithm what fraud looks like — it found natural groupings itself. Transactions that don't belong to any normal cluster are flagged automatically, catching novel fraud patterns rule-based systems miss entirely.
Reinforcement learning optimising production line, Germany
🏭 Manufacturing
Reinforcement learning optimises production scheduling. A model tried 10 million scheduling combinations in simulation, scoring each by throughput and penalising downtime. It now consistently outperforms human schedulers by 15–23%.
AI-powered personalisation, London retail flagship store
🛍️ Retail
Transfer learning powers recommendation engines. A model trained on 300 million product interactions is fine-tuned with local data. A new regional retailer can deploy a world-class recommendation system in days, not years.
Adaptive learning platform deployment, corporate HQ
👥 HR & L&D
Supervised learning predicts learner dropout risk. Labelled data trains a model that monitors reading time and quiz patterns. At-risk learners receive personalised nudges — one platform saw a 34% dropout reduction in 90 days.

📖 Key Terms
Supervised Learning +
Training a model on labelled input-output pairs so it learns to predict outputs for new inputs.
Unsupervised Learning +
Finding hidden structure in unlabelled data through clustering or dimensionality reduction.
Reinforcement Learning +
An agent learns by taking actions in an environment, maximising cumulative rewards over time.
Backpropagation +
The algorithm that adjusts model weights by calculating gradients to reduce prediction error.
Training Data +
The dataset used to fit a model's parameters — labelled for supervised, unlabelled for unsupervised.
